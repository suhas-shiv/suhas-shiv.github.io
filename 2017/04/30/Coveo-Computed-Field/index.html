Coveo Computed Field

Posted on April 30, 2017

Computed fields allow you to index values calculated while indexing. Adding fields to an index can improve runtime performance by making data available in the index rather than requiring a visit to a data source, such as an item in a Sitecore database.

If you want to do some manipulation before it gets added to index on any Sitecore field, you need to go with Coveo computed field to achieve the same. Below are the steps you can refer to have a computed field in Coveo index

How to Create Coveo Computed field for Index

  • Once you have created the computed field class, you need to update the Coveo custom config file - Coveo.SearchProvider.Custom.config as below

    Class Path - <YourProject>.ComputedFields.ComputedField (above example class)

  • Now all set to build the index, go ahead and start coveo_web_index
