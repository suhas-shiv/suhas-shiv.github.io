Coveo Searchable Field

Below are the steps to follow to make Coveo field as searchable. I have taken example of computed field (indexValue) created in my previous blog. Please refer to my previous blog on Coveo Computed Field for more info.

Following attributes are added to the Coveo.SearchProvider.Custom.config file to make any field searchable.

  • includeForFreeTextSearch attribute should be set to “True” to make a field searchable.

  • isFacet attribute should be set to “True” if you want this field to be facet(which can be used in Coveo facet interface)

Note: Free Text Search can be enabled for any field from the Coveo Admin Tool, but it will not persist longer. Once Coveo is restarted, the config changes saved from Admin Tool will be lost. Hence, it is preferred adding free text attribute to Coveo custom config file.
